定义广播数据（厂家数据）：

| 字节序 | 字段                                                         | 备注           |
| ------ | ------------------------------------------------------------ | -------------- |
| 0      | 特殊标识                                                     | 0xFE           |
| 1      | 协议版本                                                     |                |
| [2，3] | 广播间隔                                                     | 单位：0.625 ms |
| 4      | 电量                                                         | 百分比         |
| 5      | 楼层数                                                       |                |
| 6      | 开关序号                                                     |                |
| 7      | 设备状态：<br />0：出厂状态（0：初始，已安装）<br />1：舵机开关状态（0：关闭，1：开启）<br />2：红外状态（0：关闭，1：开启）<br />3：指示灯开关状态（0：关闭，1：开启） |                |
| 8      | 舵机开角度                                                   | 0~180          |
| 9      | 舵机关角度                                                   | 0~180          |
| 10     | 指示灯任务                                                   |                |

设置指令：

| 字节序 | 字段           | 备注（0xFF不设置） |
| ------ | -------------- | ------------------ |
| [0,1]  | 头             | 0xFEFE             |
| [2,3]  | 广播间隔       |                    |
| 4      | 指示灯是否开启 | 0：关闭，1：开启   |
| 5      | 指示灯任务     |                    |
| 6      | 楼层           |                    |
| 7      | 开关序号       |                    |
| 8      | 舵机开角度     |                    |
| 9      | 舵机关角度     |                    |
| 10     | 舵机开关       | 0：关闭，1：开启   |
| 11     | checkSum       | [2,10] 字节累加    |
